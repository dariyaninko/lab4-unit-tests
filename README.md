# Лабораторная работа №4: Unit-тестирование

## Описание проекта
Проект демонстрирует создание unit-тестов для модуля вычисления площади и периметра прямоугольника с использованием фреймворка `unittest`.

## Цели работы
1. Изучить основы модульного тестирования (unit testing)
2. Освоить работу с фреймворком `unittest`
3. Научиться создавать тесты для проверки корректности работы функций
4. Отработать навыки тестирования граничных случаев и исключительных ситуаций
